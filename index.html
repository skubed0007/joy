Some Projects of Mine

Neit

Neit is a high-level, simple, and easy-to-use compiled programming language. Originally started as a better way to write C, it has evolved into a language aimed at making development faster and more efficient. The goal of Neit is to simplify syntax while maintaining the power and flexibility required for complex programming tasks. It’s written in Rust for speed and reliability.

Find it on GitHub: Here

Clua

Clua is a Lua compiler that aims to integrate built-in features for enhanced ease of use. It is designed to make Lua programming simpler and more efficient. Clua is still under heavy development and is not ready to use yet It’s built using Rust for performance.

Find it on GitHub: Here

Cutils

Cutils is a project focused on creating a simplified, more efficient version of coreutils. It’s a coreutils rewrite, designed to make the command-line experience faster and more intuitive. While it won’t replace the original coreutils, it aims to offer an optimized alternative. The project is written in C, but I plan to transition it to Rust for better memory safety and performance.

Find it on GitHub: Here

CProjMaker

CProjMaker is a tool that helps developers create C++ projects easily and quickly. It’s a straightforward project maker for C++, currently in a basic form. I have plans to overhaul and rewrite the tool, with the intention to shift the development to Rust to improve performance and add new features.
